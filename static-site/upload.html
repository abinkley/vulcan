Upload Background Image

To use the screenshot as the background image:

  1. Upload the screenshot image using the form below
  2. Rename the file to vulcan-screenshot-bg.jpg
  3. Place it in the images directory of your repository
  4. Update the URL in index.html to point to this file
  5. Deploy the site again

Note: This form will redirect you to GitHub. You'll need to be logged in to your GitHub account to complete the upload.
